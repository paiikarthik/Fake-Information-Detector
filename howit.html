Verification Pipeline

How This Project Checks Whether Information Is Real

FakeXpose AI does not depend on one search result or one AI answer. It extracts the claim, searches news and fact-check sources, compares meaning with semantic similarity, scores source credibility, and then returns a verdict with confidence.

Why Gemini Is Used

Gemini is used for claim extraction. When the user pastes a long paragraph, Gemini converts it into structured data: main claims, important keywords, and named entities. This makes the search query cleaner and helps the system verify the exact statement instead of searching the whole paragraph blindly.

Why NewsAPI Is Used

NewsAPI retrieves recent related news articles from multiple publishers. If a NewsAPI key is not available or fails, the project falls back to Google News RSS so verification can still continue with public news results.

Why Fact-Check Sites Matter

The project separately searches trusted fact-checking domains. These results get top priority because they are written specifically to confirm, debunk, or explain viral claims.

Project Flow

1

Claim Extraction

Gemini extracts claims, keywords, and entities. If Gemini is not configured, the app uses local NLTK processing as a fallback.

2

News Retrieval

The app searches NewsAPI for related articles. If NewsAPI is unavailable, it searches Google News RSS.

3

Fact-Check Retrieval

The app searches official fact-check sources: Snopes, PolitiFact, FactCheck.org, and Reuters Fact Check.

4

Semantic Similarity

SentenceTransformer compares the extracted claim with article titles. This checks whether the article is actually about the same meaning, not just matching a few words.

5

Credibility Scoring

Each source gets a trust score. Official fact-checkers and major established publishers score higher. Blogs, forums, social media, and low-trust platforms are heavily reduced.

6

Final Verdict

The decision engine combines fact-check stance, semantic similarity, source credibility, refutation words, and critical event alignment to produce VERIFIED, FALSE, or UNVERIFIED.

How Accuracy Is Improved

Fact-check priority

If a relevant Snopes, PolitiFact, FactCheck.org, or Reuters Fact Check result is found, it has priority over ordinary news results.

Meaning-based matching

The system uses semantic similarity, so a claim can match an article even when the wording is different.

Refutation detection

Words like false, fake, misleading, hoax, debunked, warning, harmful, and untrue help detect when an article is actually refuting the claim.

Critical event checks

For sensitive events like death, arrest, or resignation, the article title must align with that same event type. This reduces false positives from articles about the same person but a different event.

Important: this project improves verification by combining multiple signals, but no automated system is perfect. Users should still read the linked sources for high-stakes claims.
